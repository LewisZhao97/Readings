## Useful Tools for Agent-Obsidian Knowledge Base Building
---
### [Obsidian Web Clipper](https://obsidian.md/clipper)
Official Obsidian browser extension. Clips almost any text or image-based page directly into the Obsidian desktop app.

---
### [Reader](https://github.com/jina-ai/reader)
Open-source tool that converts any URL into LLM-friendly input with a simple prefix: [https://r.jina.ai/](https://r.jina.ai/)

---
### [MarkItDown](https://github.com/microsoft/markitdown)
Open-source Python tool from Microsoft for converting files and office documents to Markdown.
